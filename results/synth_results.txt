=== Notes ===
the filter is tiny. less than 1% of the FPGA. the BRAM is a single 18Kb
primitive configured as 8-bit wide x 2048 deep (we only use 64 depth, 
so the rest is wasted, but you cant allocate half a BRAM).

my supervisor was right. hardware is cheap. doing this in software took
a whole C program and still had microsecond jitter. the hardware version
uses basically zero resources and guarantees 10ns deterministic latency.
