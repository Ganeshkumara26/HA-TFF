# Synthesis and Implementation Results

This folder contains the raw reports generated by Xilinx Vivado 2026.1 for the HA-TFF (Hardware-Accelerated Telemetry Filter) module targeting the Artix-7 (xc7a100tcsg324-1) FPGA.

## Highlights
- **Utilization**: Extremely lightweight. Only 613 Slice LUTs (0.97%) and 962 Registers (0.76%).
- **Timing**: Fully passes all timing constraints at 100 MHz (10ns period). Worst Negative Slack (WNS) is 4.706ns, meaning the logic could comfortably scale to 180+ MHz if needed.
- **Latency**: True line-rate processing. Packets are parsed and forwarded to the Wishbone bus with a determinable 190ns latency (19 cycles at 100 MHz).

The low utilization confirms that this module can be seamlessly integrated into the overarching SoC alongside the VeeR EH1 core without causing routing congestion.
